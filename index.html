HTML

  • First Computer Language: Hypertext Markup Language
  • Builds the structure of a webpage
  • Elements types in HTML: text, image, header, list which are seen elements
  • Elements types in HTML: meta related, content division, semantic divide and describe the page
  • The head element contains information about the webpage
  • Metadata tags used in head element: title, style, base, link, meta, and script
  • The body element represents the visible content shown to the user
  • The body element defines the body of the document and is an example of a semantic HTML
  • The name of an element in the body describes its content
  • Content of the body is contained between opening and closing tag with angle brackets
  • h represents heading
  • p represents paragraph or block of text
  • ul represents unordered list
  • ol represents ordered list
  • li represents list item
  • img represents image
  • The section elements are semantic elements that define a section in a document
  • section elements allow to elegantly organize data

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indixates how much space we want around the content inside an element
  • A CSS rule can be based on an element name
  • A selector defines the element or attributes to which the rules will apply
  • Declarations contain two important components: the CSS property and the value of the property
  • A CSS rule can also be assigned to a class attribute with a class selector
  • A link element creates the connection to an external CSS file
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS, and external CSS style sheet

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git merge: merge a specified branch's history into the current one
  • git pull: fetch and merge any commits from the tracking remote branch

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
