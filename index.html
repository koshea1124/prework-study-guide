CSS

  • Second Computer Language: Cascading Style Sheets (CSS)
  • Programming language used by developers to define how webpage conntent is presented to users
  • Defines colors, aesthetics, viual layout, and organization
  • 3 ways to sytle a webpage using CSS: inline CSS, internal CSS stylesheet, and external CSS stylesheet
  • Inline CSS styles an elemnt in an HTML file
  • An entire stylesheet can be embedded in an HTML file: internnal stylesheet
  • Preferred method is external CSS stylesheet
  • CSS rules are stored in a separate file which makes design changes easier
  • A margin indicates how much space we want around the outside of an element
  • A padding indixates how much space we want around the content inside an element
  • A CSS rule can be based on an element name
  • A selector defines the element or attributes to which the rules will apply
  • Declarations contain two important components: the CSS property and the value of the property
  • A CSS rule can also be assigned to a class attribute with a class selector
  • The period preceding card designates a class selector
  • A link element creates the connection to an external CSS file
