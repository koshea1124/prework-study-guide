HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The section elements are semantic elements that define a section in a document

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indixates how much space we want around the content inside an element
  • A CSS rule can be based on an element name
  • A selector defines the element or attributes to which the rules will apply
  • Declarations contain two important components: the CSS property and the value of the property
  • A CSS rule can also be assigned to a class attribute with a class selector
  • A link element creates the connection to an external CSS file
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS, and external CSS style sheet

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git merge: merge a specified branch's history into the current one
  • git pull: fetch and merge any commits from the tracking remote branch
